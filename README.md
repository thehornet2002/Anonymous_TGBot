# Tel2Anon

یک ربات تلگرام کاملاً متن‌باز (Open Source) برای دریافت پیام ناشناس، که کاملاً روی **Cloudflare Workers** اجرا می‌شود — بدون هیچ سروری، بدون هیچ پایگاه‌داده‌ای.

هر کسی پیامی به ربات بفرستد، بدون هیچ نام یا شناسه‌ای برای شما (مالک ربات) فوروارد می‌شود. شما هم می‌توانید با زدن دکمه، به‌صورت ناشناس پاسخ بدهید.

## طراحی حریم خصوصی

- هیچ دیتابیس، KV یا D1‌ای در این پروژه استفاده نشده. کد کاملاً **stateless** است.
- آیدی عددی فرستنده هیچ‌جا در سمت سرور (Worker) ذخیره نمی‌شود و در هیچ لاگی نوشته نمی‌شود.
- برای این‌که امکان «پاسخ ناشناس» هم وجود داشته باشد، آیدی فرستنده فقط و فقط به‌صورت موقت داخل همان پیام تلگرامی که خود شما (مالک) می‌بینید قرار می‌گیرد (پیام «✍️ پاسخ خود را بنویسید...ref:12345»). این پیام فقط در چت خصوصی خودتان با ربات است و جای دیگری ذخیره نمی‌شود.
- اگر حتی همین مقدار را هم نمی‌خواهید (یعنی می‌خواهید ربات کاملاً یک‌طرفه باشد و اصلاً امکان پاسخ نداشته باشد)، کافی‌ست بخش `reply_markup` در تابع `handleUpdate` و کل تابع `handleCallback` را حذف کنید — چند خط کد است و در فایل `src/index.js` مشخص شده.

## پیش‌نیازها

- یک اکانت Cloudflare (رایگان کافی است)
- Node.js نصب‌شده روی سیستم
- یک توکن ربات از [@BotFather](https://t.me/BotFather)

## مراحل راه‌اندازی

### ۱. ساخت ربات در تلگرام

در تلگرام به [@BotFather](https://t.me/BotFather) پیام دهید:

```
/newbot
```

و مراحل را طی کنید تا یک توکن مثل `123456789:AAExample...` بگیرید.

### ۲. نصب Wrangler و ورود به حساب Cloudflare

```bash
npm install -g wrangler
wrangler login
```

### ۳. تنظیم متغیرهای محرمانه (Secrets)

هیچ‌کدام از این مقادیر داخل کد یا `wrangler.toml` نوشته نمی‌شوند؛ Cloudflare آن‌ها را رمزنگاری‌شده نگه می‌دارد:

```bash
wrangler secret put BOT_TOKEN
# توکنی که از BotFather گرفتید را وارد کنید

wrangler secret put OWNER_ID
# آیدی عددی خودتان، مثلاً: 8101836386

wrangler secret put WEBHOOK_SECRET
# یک رشته‌ی تصادفی دلخواه، مثلاً خروجی:  openssl rand -hex 24
```

### ۴. دیپلوی روی Cloudflare

```bash
wrangler deploy
```

در خروجی، آدرسی شبیه این می‌گیرید:

```
https://tel2anon-bot.<your-subdomain>.workers.dev
```

### ۵. ثبت Webhook در تلگرام

با همان `WEBHOOK_SECRET` که در مرحله ۳ ساختید:

```bash
curl -X POST "https://api.telegram.org/bot<BOT_TOKEN>/setWebhook" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://tel2anon-bot.<your-subdomain>.workers.dev",
    "secret_token": "<WEBHOOK_SECRET>"
  }'
```

اگر پاسخ `{"ok":true,...}` گرفتید، ربات فعال است.

### ۶. تست

- خودتان (مالک) به ربات پیام `/start` بدهید تا پیام خوش‌آمد مدیر را ببینید.
- از یک اکانت دیگر (یا از دوستتان بخواهید) به ربات پیام بدهد — باید بدون نام و آیدی برایتان فوروارد شود.
- روی دکمه «↩️ پاسخ ناشناس» زیر آن پیام بزنید، جواب را تایپ کنید و بفرستید — به‌صورت ناشناس برای طرف مقابل ارسال می‌شود.

```
tel2anon-bot/

├── LICENSE             # MIT

```

## نکات

- ربات فقط در چت خصوصی (Private) کار می‌کند؛ پیام‌های گروه/کانال نادیده گرفته می‌شوند.
- می‌توانید تمام متن‌های فارسی داخل آبجکت `TEXT` در ابتدای `src/index.js` را به دلخواه تغییر دهید.
- برای مشاهده لاگ‌های زنده: `wrangler tail`
